# Арканойд

Тестовое задание.

## Технологии
- HTML5
- CSS3
- JavaScript

## Реализовано
- управление платформой
- физика движения шарика
- столкновения
- система жизней
- счёт
- ускорение шарика
- логика победы и поражения

## Обигре
- Управление платформой возможно только стрелками в лево в право (⬅ ➡)
- Начало игры и после потери жизни что бы продолжить нажмине (⬅ ➡)
- за каждый блок начисляеться 100 очков
- за каждый блок скорость шарика возрастает, но при потери жизни сбрасывается на начльное значение
